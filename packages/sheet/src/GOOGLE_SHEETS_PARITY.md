# Sheet — Google Sheets 호환 로드맵 / Task List

> 이 문서는 `Sheet` 컴포넌트를 Google Sheets와 최대한 호환되도록 확장하기 위한
> **작업 목록**입니다. 각 태스크는 독립적으로 착수 가능하도록 목표·참조 동작·구현

  ⌘X(잘라내기)는 `onGridCopy`를 재사용해 클립보드에 쓴 뒤 원본 셀의 값·서식을
  모두 비움(엑셀처럼 서식까지 함께 이동). ⌘⇧V(값만 붙여넣기)는 네이티브 paste
  이벤트가 없는 키보드 단축키라서, 복사/잘라내기 시점에 이미 평가해둔 값을
  담아두는 내부 섀도 클립보드(`internalClipboardRef`)를 읽어 서식·수식 없이 계산된
  값만 붙여넣음(수식 셀을 복사했다면 수식 텍스트가 아니라 그 결과 숫자/문자열이
  그대로 붙음).
- [x] T11. 함수 확장 — `engine.ts`를 값 모델(`number | string | boolean`)과 지연평가
  (thunk) 기반으로 재작성. 비교연산자(`= <> < > <= >=`), 문자열 리터럴(`"..."`,
  `""` 이스케이프), 연결연산자 `&` 추가. 함수: `IF/IFERROR`(지연 분기 — 선택 안 된
  브랜치는 평가하지 않음), `AND/OR/NOT`, `ROUND/FLOOR/CEILING`(significance 인자,
  진짜 엑셀 방식), `ABS/MOD`, `COUNTA/COUNTIF/SUMIF`(와일드카드 대신 `">10"`류
  비교 접두사 지원), `CONCATENATE/LEFT/RIGHT/MID/LEN`, `VLOOKUP/HLOOKUP/INDEX/MATCH`
  (근사/정확 매치, range를 `{addrs,nrows,ncols}` 사각 매트릭스로 취급),
  `TODAY/NOW/DATE`(날짜 직렬값 없이 `"YYYY-MM-DD[ HH:MM:SS]"` 문자열로 단순화 —
  날짜 산술은 지원하지 않음, 알려진 한계). 에러는 Excel 표기(`#DIV/0!` `#N/A` `#REF!`
  `#VALUE!` `#NAME?`)로 던지고 `evaluateCell`에서 문자열로 표면화(기존 `#CYCLE`는
  순환참조 전용으로 유지).
  ⚠️ 구현 중 겪은 TS 내로잉 함정: `if (!x || x.kind !== 'range') fail('...');`처럼
  가드 뒤에 `x`를 계속 좁혀 쓰려면 `fail(...)`을 **문(statement)으로 호출하지 말고
  `return fail(...)`로 호출**해야 함 — bare 호출은 `never` 반환 타입이어도 TS
  control-flow narrowing이 적용되지 않는 경우가 있어 이후 `x.addrs`/`x.get()` 접근이
  "possibly undefined"로 계속 에러났음. 모든 lookup류 가드를 `return fail(...)`로
  바꾸자 캐스트(`as Extract<ArgNode, ...>`) 없이 자연스럽게 좁혀짐.
- [x] T12. 상대/절대 참조 & 채우기 시 참조 이동 — `$A$1`/`A$1`/`$A1` 파싱
  (`CELL_RE`가 `$` 캡처), `shiftFormula(raw, dCol, dRow)`가 수식 안의 모든 상대
  참조만 델타만큼 이동(절대 축은 고정, 문자열 리터럴 안의 텍스트는 건드리지 않음 —
  따옴표 구간을 정규식에서 별도 매치해 스킵). 채우기 핸들이 단일 셀 수식을
  cyclic-copy할 때(`makeSeriesExtender`가 수식엔 관여하지 않으므로 항상 이 경로)
  `shiftFormula`를 호출해 참조를 이동. 복사/붙여넣기는 `onGridCopy`가 복사 원점을
  커스텀 MIME(`application/x-softium-sheet-origin`)에 함께 실어 보내고,
  `onGridPaste`가 (붙여넣기 앵커 − 원점) 델타로 블록 전체를 한 번에 이동 — 원점
  MIME이 없으면(외부 클립보드에서 온 텍스트) 수식이어도 그대로 붙여넣어 시프트하지
  않음(원본 위치를 알 수 없으므로).
  실제 검증: `=C2*D2`가 있는 셀을 채우기 핸들로 5행 아래로 드래그 → `=C7*D7`로
  정확히 이동(값도 참조된 빈 셀에 맞춰 0으로 재계산); 같은 셀을 복사해 다른 행에
  붙여넣기 → 붙여넣은 위치 기준으로 재이동됨을 raw 수식 텍스트로 직접 확인.
- [x] 툴바 함수 삽입(Σ) — `SUPPORTED_FUNCTIONS`(engine.ts, 함수명+힌트 목록) 기반
  드롭다운. 클릭하면 활성 셀 편집기가 `=FUNC()`로 열리고 캐럿이 괄호 사이에 위치
  (`pendingCaretRef` + 편집기 마운트 후 `setSelectionRange`), 바로 범위를 타이핑할
  수 있음.

## 툴바 구성 참조 (Google Sheets 순서)

```
[실행취소][다시실행] | [인쇄][서식복사] | [줌] |
[통화][%][.00→][.0←] | [숫자형식▾] |
[글꼴▾][크기] |
[B][I][U][S] | [글자색][채우기색] |
[테두리▾][병합▾] |
[가로정렬▾][세로정렬▾][줄바꿈▾][회전▾] |
[링크][댓글][차트][필터][함수Σ]
```

현재 구현: `실행취소/다시실행`, `B/I/U/S`, `글자색/채우기색`, `가로정렬`, `세로정렬/줄바꿈`,
`통화/%/.00↔.0/숫자형식▾`, `글꼴▾/크기`, `테두리▾/병합▾`, `서식복사/서식지우기`, `함수Σ`
(SUPPORTED_FUNCTIONS 드롭다운 — `=FUNC()` 삽입 후 괄호 사이로 캐럿 이동).
남은 것(줌, 인쇄, 링크/댓글/차트/필터, 회전)은 위 T13–T20 태스크로 채운다.

## 작업 규칙

1. 한 번에 한 태스크(`[ ]`). 완료 시 `[x]`로 바꾸고 "구현 완료" 목록에도 반영.
2. `CellFormat`/상태 추가 시 undo 스냅샷(T10)과 셀 렌더 반영을 함께.
3. 항상 타입체크 → 빌드 → 프리뷰(`SheetPage`)에서 실제 동작 검증(스크린샷).
4. 스타일은 토큰(`var(--sft-*)`)만 사용, 하드코딩 색상 금지.
5. 접근성: 버튼 `aria-label`, 키보드 접근 유지.
