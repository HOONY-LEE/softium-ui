> **작업 목록**입니다. 각 태스크는 독립적으로 착수 가능하도록 목표·참조 동작·구현

## 우선순위 1 — 서식 & 편집 코어

전부 완료 (T1–T6, 상단 "현재 상태" 참고). 남은 세부 보강 아이디어:
- T3 wrap: 현재는 줄바꿈만 지원 — 내용에 맞춘 행 높이 auto-grow는 아직 없음(측정 후
  `rowHeights` 갱신 필요, 후순위).
- T4 borders: 두께/스타일(dashed/dotted)은 타입상 지원하나 툴바에 색상·두께 커스터마이즈
  UI는 없음(프리셋 4종만).
- T6 페인트: 현재 클릭 한 셀에만 적용(드래그 범위 적용은 미지원).

- **참조**: ⌘X/⌘C/⌘V, 우클릭 메뉴. 외부(엑셀/시트)와 TSV 상호운용.

  "값만 붙여넣기"(⌘⇧V) 옵션.
- **수용**: 시트 내부 복붙 시 값·서식 보존; 엑셀에서 복사한 TSV 붙여넣기 동작.

### [ ] T8. 우클릭 컨텍스트 메뉴
- **구현**: 셀/헤더 우클릭 시 메뉴(잘라/복사/붙여넣기, 행·열 삽입/삭제, 지우기,
  서식, 병합 등). 위치 계산 + 외부 클릭 닫기.
- **수용**: 각 항목이 해당 동작 트리거.

### [ ] T9. 행/열 삽입·삭제·이동

- **수용**: 3행 삭제 시 4행↑이 위로 당겨지고 수식 참조 정정.

## 우선순위 3 — 수식 & 분석

### [ ] T11. 함수 확장
- **현재**: SUM/AVERAGE/MIN/MAX/COUNT.
- **구현**(`engine.ts`): IF, AND/OR/NOT, ROUND/FLOOR/CEIL, ABS, MOD, COUNTA,
  COUNTIF/SUMIF, CONCATENATE/`&`, LEFT/RIGHT/MID/LEN, VLOOKUP/HLOOKUP, INDEX/MATCH,
  TODAY/NOW/DATE, IFERROR. 비교연산자(`= <> < > <= >=`)·문자열 리터럴 토크나이저 확장.
- **수용**: 각 함수 단위 동작 + 중첩/에러(`#REF!`, `#DIV/0!`, `#N/A`) 표현.

### [ ] T12. 상대/절대 참조 & 채우기 시 참조 이동
- **구현**: `$A$1`/`A$1`/`$A1` 파싱. 채우기 핸들·복붙 시 상대 참조는 오프셋만큼 이동,
  절대 참조는 고정. 현재 시리즈 자동채우기(`makeSeriesExtender`)와 통합 —
  수식 셀은 값 반복이 아니라 참조 시프트로 채워야 함.
- **수용**: `=A1`을 아래로 채우면 `=A2, =A3…`; `=$A$1`은 고정.

### [ ] T13. 정렬 & 필터
- **구현**: 범위/열 정렬(오름/내림), 필터 뷰(헤더 드롭다운으로 값 체크·조건).
  정렬은 행 단위 재배치(값+서식 함께 이동), 수식 참조 처리 주의.
- **수용**: 열 기준 정렬 시 행 전체가 함께 정렬; 필터로 행 숨김.

### [ ] T14. 데이터 유효성 검사 (드롭다운/체크박스)
- **구현**: 셀에 유효성 규칙(목록/범위/불리언). 목록이면 셀에 드롭다운 chip,
  체크박스면 토글. `CellFormat` 또는 별도 `validations` 상태.
- **수용**: 드롭다운 목록 선택, 체크박스 토글.

## 우선순위 4 — 뷰 & 협업 UX

### [ ] T15. 틀 고정 (Freeze rows/cols)
- **구현**: `frozenRows/frozenCols` 상태. 해당 행/열을 `position: sticky`로 고정,
  스크롤 시 유지. 오프셋/오버레이 계산에 반영.
- **수용**: 1행 고정 후 세로 스크롤에도 헤더 데이터 유지.

### [ ] T16. 확대/축소 (Zoom)
- **구현**: 툴바 줌 셀렉트(50–200%). 격자에 `transform: scale` 또는 폰트/치수 스케일.
  좌표 계산 스케일 반영.
- **수용**: 줌 변경 시 선택/리사이즈 좌표 정확.

### [ ] T17. 조건부 서식
- **구현**: 규칙(값 조건/색상 스케일) → 렌더 시 동적 배경/글자색. 규칙 관리 패널.
- **수용**: "값 > 100 이면 빨강" 규칙이 실시간 반영.

### [ ] T18. 찾기 및 바꾸기 (⌘F / ⌘⇧H)
- **구현**: 오버레이 검색, 매치 하이라이트/이동, 바꾸기/모두 바꾸기(대소문자·정규식 옵션).
- **수용**: 검색 이동 + 치환 동작.

### [ ] T19. 셀 메모/댓글
- **구현**: 셀에 노트(`notes[addr]`), 우상단 삼각 표식 + 호버 팝오버.
- **수용**: 메모 추가/편집/삭제, 표식 표시.

### [ ] T20. 하이퍼링크 / 이미지 / 차트 (선택)
- **구현**: 링크는 `=HYPERLINK()` 또는 서식. 차트/이미지는 별도 오버레이 레이어
  (대규모, 후순위).

## 툴바 구성 참조 (Google Sheets 순서)

```
[실행취소][다시실행] | [인쇄][서식복사] | [줌] |
[통화][%][.00→][.0←] | [숫자형식▾] |
[글꼴▾][크기] |
[B][I][U][S] | [글자색][채우기색] |
[테두리▾][병합▾] |
[가로정렬▾][세로정렬▾][줄바꿈▾][회전▾] |
[링크][댓글][차트][필터][함수Σ]
```

현재 구현: `실행취소/다시실행`, `B/I/U/S`, `글자색/채우기색`, `가로정렬`, `세로정렬/줄바꿈`,
`통화/%/.00↔.0/숫자형식▾`, `글꼴▾/크기`, `테두리▾/병합▾`, `서식복사/서식지우기`.
남은 것(줌, 인쇄, 링크/댓글/차트/필터/함수Σ, 회전)은 위 T7–T20 태스크로 채운다.

## 작업 규칙

1. 한 번에 한 태스크(`[ ]`). 완료 시 `[x]`로 바꾸고 "구현 완료" 목록에도 반영.
2. `CellFormat`/상태 추가 시 undo 스냅샷(T10)과 셀 렌더 반영을 함께.
3. 항상 타입체크 → 빌드 → 프리뷰(`SheetPage`)에서 실제 동작 검증(스크린샷).
4. 스타일은 토큰(`var(--sft-*)`)만 사용, 하드코딩 색상 금지.
5. 접근성: 버튼 `aria-label`, 키보드 접근 유지.
