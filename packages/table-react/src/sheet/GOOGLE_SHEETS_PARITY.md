## 우선순위 1 — 서식 & 편집 코어

### [ ] T1. 숫자 서식 (Number formats)
- **참조**: Google Sheets 툴바의 통화($), 백분율(%), 소수 자리수 증감, "형식 > 숫자"
- **동작**: 셀 값은 원본(raw) 유지하고 표시(display)만 포맷. 통화/백분율/천단위
  콤마/소수 자릿수/날짜/시간/커스텀 패턴.
- **구현**:
  - `CellFormat`에 `numFmt?: string`(예: `'#,##0'`, `'0.00%'`, `'$#,##0.00'`, `'yyyy-mm-dd'`) 추가.
  - `engine.ts`에 `formatNumber(value, pattern)` 추가(자체 구현 또는 경량 포맷터).
  - 셀 렌더에서 `typeof value === 'number' && fmt.numFmt`이면 포맷 적용.
  - 툴바에 통화/백분율/소수증감 버튼 그룹 + "더보기" 드롭다운.
- **수용**: `1234.5` + `'#,##0.00'` → `1,234.50`; `0.25` + `0.00%` → `25.00%`.

### [ ] T2. 글꼴 & 글자 크기
- **참조**: 툴바 좌측 글꼴 드롭다운, 크기 숫자 스텝퍼.
- **구현**: `CellFormat`에 `fontFamily?: string`, `fontSize?: number`. 툴바에 셀렉트/스텝퍼.
  셀 `contentStyle`에 반영. 행 높이 auto-grow 고려(선택).
- **수용**: 선택 셀에 글꼴/크기 적용, 앵커 셀 기준 툴바 값 동기화.

### [ ] T3. 세로 정렬 & 텍스트 줄바꿈
- **구현**: `CellFormat`에 `valign?: 'top'|'middle'|'bottom'`, `wrap?: 'overflow'|'wrap'|'clip'`.
  셀 flex `align-items` + `.sft-sheet__value`의 `white-space`/`overflow` 처리.
  wrap이면 행 높이가 내용에 맞게 늘어나야 함(측정 후 `rowHeights` 갱신 또는 자동 높이 모드).
- **수용**: 긴 텍스트 wrap 시 셀 안에서 줄바꿈 + 행 높이 증가; clip 시 잘림.

### [ ] T4. 테두리 (Borders)
- **참조**: 툴바 테두리 드롭다운(전체/바깥/안쪽/상하좌우/없음, 색/두께/스타일).
- **구현**: `CellFormat`에 `borders?: {top?,right?,bottom?,left?: BorderSpec}`.
  플로팅 오버레이 대신 셀 `box-shadow`/`border`로 그리되 인접 셀 경계 겹침 처리 주의.
  드롭다운 UI 신설.
- **수용**: 범위 바깥 테두리만, 안쪽 격자만 등 각 프리셋 동작.

### [ ] T5. 셀 병합 (Merge cells)
- **참조**: 툴바 병합 버튼(모두/가로/세로/해제).
- **구현**: `merges: {r,c,rowSpan,colSpan}[]` 상태. 렌더에서 병합 앵커 셀만 그리고
  `colSpan/rowSpan`만큼 width/height 확장, 가려지는 셀은 skip. 선택/복사/수식 참조 시
  병합 영역을 단일 셀로 취급. 리사이즈/오프셋 계산에 반영.
- **수용**: 2×2 병합 후 단일 셀처럼 선택·편집, 해제 시 원복.

### [ ] T6. 서식 지우기 / 서식 복사(페인트)
- **구현**: "서식 지우기"(⌘\)는 선택 범위 `formats`에서 제거. 페인트 포맷 버튼은
  앵커 서식을 클립보드처럼 들고 다음 클릭/드래그 범위에 적용.
- **수용**: 서식 복사 후 다른 범위에 붙으면 동일 서식 적용.

## 우선순위 2 — 클립보드 & 데이터

### [ ] T7. 잘라내기 / 복사 / 붙여넣기 (values + formats)
- **참조**: ⌘X/⌘C/⌘V, 우클릭 메뉴. 외부(엑셀/시트)와 TSV 상호운용.
- **구현**: 선택 범위를 TSV(값) + 내부 JSON(서식)으로 클립보드에 기록.
  붙여넣기 시 `navigator.clipboard`에서 TSV 파싱해 앵커 기준 확장. 잘라내기는 복사+삭제.
  "값만 붙여넣기"(⌘⇧V) 옵션.
- **수용**: 시트 내부 복붙 시 값·서식 보존; 엑셀에서 복사한 TSV 붙여넣기 동작.

### [ ] T8. 우클릭 컨텍스트 메뉴
- **구현**: 셀/헤더 우클릭 시 메뉴(잘라/복사/붙여넣기, 행·열 삽입/삭제, 지우기,
  서식, 병합 등). 위치 계산 + 외부 클릭 닫기.
- **수용**: 각 항목이 해당 동작 트리거.

### [ ] T9. 행/열 삽입·삭제·이동
- **구현**: 헤더 컨텍스트 메뉴 또는 툴바. 삽입/삭제 시 `cells`/`formats` 주소 리맵
  (해당 인덱스 이상 시프트), 수식 참조도 이동 반영(엔진 리라이트). 드래그로 열/행 이동.
- **수용**: 3행 삭제 시 4행↑이 위로 당겨지고 수식 참조 정정.

### [ ] T10. 실행취소 범위 확장
- **현재**: undo/redo는 `cells` + `formats`만 스냅샷.
- **구현**: `colWidths/rowHeights/rowCount/colCount/colAlign/merges` 등 구조 변경도
  히스토리에 포함. `snapshot()`이 전체 문서 스냅샷을 찍도록 확장하고 모든 구조 변경
  액션에서 호출.
- **수용**: 열 리사이즈·행 추가·병합도 undo/redo.

## 우선순위 3 — 수식 & 분석

### [ ] T11. 함수 확장
- **현재**: SUM/AVERAGE/MIN/MAX/COUNT.
- **구현**(`engine.ts`): IF, AND/OR/NOT, ROUND/FLOOR/CEIL, ABS, MOD, COUNTA,
  COUNTIF/SUMIF, CONCATENATE/`&`, LEFT/RIGHT/MID/LEN, VLOOKUP/HLOOKUP, INDEX/MATCH,
  TODAY/NOW/DATE, IFERROR. 비교연산자(`= <> < > <= >=`)·문자열 리터럴 토크나이저 확장.
- **수용**: 각 함수 단위 동작 + 중첩/에러(`#REF!`, `#DIV/0!`, `#N/A`) 표현.

### [ ] T12. 상대/절대 참조 & 채우기 시 참조 이동
- **구현**: `$A$1`/`A$1`/`$A1` 파싱. 채우기 핸들·복붙 시 상대 참조는 오프셋만큼 이동,
  절대 참조는 고정. 현재 시리즈 자동채우기(`makeSeriesExtender`)와 통합 —
  수식 셀은 값 반복이 아니라 참조 시프트로 채워야 함.
- **수용**: `=A1`을 아래로 채우면 `=A2, =A3…`; `=$A$1`은 고정.

### [ ] T13. 정렬 & 필터
- **구현**: 범위/열 정렬(오름/내림), 필터 뷰(헤더 드롭다운으로 값 체크·조건).
  정렬은 행 단위 재배치(값+서식 함께 이동), 수식 참조 처리 주의.
- **수용**: 열 기준 정렬 시 행 전체가 함께 정렬; 필터로 행 숨김.

### [ ] T14. 데이터 유효성 검사 (드롭다운/체크박스)
- **구현**: 셀에 유효성 규칙(목록/범위/불리언). 목록이면 셀에 드롭다운 chip,
  체크박스면 토글. `CellFormat` 또는 별도 `validations` 상태.
- **수용**: 드롭다운 목록 선택, 체크박스 토글.

## 우선순위 4 — 뷰 & 협업 UX

### [ ] T15. 틀 고정 (Freeze rows/cols)
- **구현**: `frozenRows/frozenCols` 상태. 해당 행/열을 `position: sticky`로 고정,
  스크롤 시 유지. 오프셋/오버레이 계산에 반영.
- **수용**: 1행 고정 후 세로 스크롤에도 헤더 데이터 유지.

### [ ] T16. 확대/축소 (Zoom)
- **구현**: 툴바 줌 셀렉트(50–200%). 격자에 `transform: scale` 또는 폰트/치수 스케일.
  좌표 계산 스케일 반영.
- **수용**: 줌 변경 시 선택/리사이즈 좌표 정확.

### [ ] T17. 조건부 서식
- **구현**: 규칙(값 조건/색상 스케일) → 렌더 시 동적 배경/글자색. 규칙 관리 패널.
- **수용**: "값 > 100 이면 빨강" 규칙이 실시간 반영.

### [ ] T18. 찾기 및 바꾸기 (⌘F / ⌘⇧H)
- **구현**: 오버레이 검색, 매치 하이라이트/이동, 바꾸기/모두 바꾸기(대소문자·정규식 옵션).
- **수용**: 검색 이동 + 치환 동작.

### [ ] T19. 셀 메모/댓글
- **구현**: 셀에 노트(`notes[addr]`), 우상단 삼각 표식 + 호버 팝오버.
- **수용**: 메모 추가/편집/삭제, 표식 표시.

### [ ] T20. 하이퍼링크 / 이미지 / 차트 (선택)
- **구현**: 링크는 `=HYPERLINK()` 또는 서식. 차트/이미지는 별도 오버레이 레이어
  (대규모, 후순위).

## 툴바 구성 참조 (Google Sheets 순서)

```
[실행취소][다시실행] | [인쇄][서식복사] | [줌] |
[통화][%][.00→][.0←] | [숫자형식▾] |
[글꼴▾][크기] |
[B][I][U][S] | [글자색][채우기색] |
[테두리▾][병합▾] |
[가로정렬▾][세로정렬▾][줄바꿈▾][회전▾] |
[링크][댓글][차트][필터][함수Σ]
```

현재 구현: `실행취소/다시실행`, `B/I/U/S`, `글자색/채우기색`, `가로정렬`.
나머지는 위 T1–T20 태스크로 채운다.

## 작업 규칙

1. 한 번에 한 태스크(`[ ]`). 완료 시 `[x]`로 바꾸고 "구현 완료" 목록에도 반영.
2. `CellFormat`/상태 추가 시 undo 스냅샷(T10)과 셀 렌더 반영을 함께.
3. 항상 타입체크 → 빌드 → 프리뷰(`SheetPage`)에서 실제 동작 검증(스크린샷).
4. 스타일은 토큰(`var(--sft-*)`)만 사용, 하드코딩 색상 금지.
5. 접근성: 버튼 `aria-label`, 키보드 접근 유지.
